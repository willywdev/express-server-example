# express-server-example
A minimal Express.js project for testing and learning purposes.

## Quick Start

1. Install dependencies:
```bash
yarn install
```

2. Start the server:
```bash
yarn start
```

3. Open `http://localhost:3000` and create/delete files

## Development

For live-reloading while developing (auto-restarts on file changes):
```bash
yarn dev
```

## Features
- Create and delete files via web interface
- Real-time file list updates
- `.gitkeep` is protected (can't delete or see it)
